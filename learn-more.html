What this demonstrates

The location shown above was captured from the QR code you scanned. The code encodes a simple numeric identifier that maps to a campus location. When you navigate across pages, that identifier is preserved in the URL using a small helper, illustrating how easily contextual data can be collected and propagated during normal browsing.

This is a minimal example of passive data collection: by following links, your context (in this case, where you scanned from) is carried along without any extra prompts.
